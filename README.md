# Editorial Superpowers

Editorial Superpowers is a lightweight skill set for publishing editors. It is designed for three high-value moments in editorial work:

1. figuring out the strongest direction before writing
2. organizing a convincing internal proposal report once the direction is mostly clear
3. preparing a reader-facing writing brief once the angle is chosen

It keeps two core skills from the original Superpowers idea, but repurposes them for publishing instead of software delivery.

## Core Skills

### `brainstorming`
Use when the work is still fuzzy.

Best for:
- book positioning
- audience exploration
- selling-point comparison
- recommendation angle exploration
- post / launch hook exploration

Typical result:
- 3-5 candidate angles
- target reader for each angle
- recommended primary angle

### `proposal-planning`
Use when the direction is mostly clear and the work now needs structure.

Best for:
- internal proposal report planning
- section planning
- writing sequence
- evidence gap listing
- organizing support materials

Typical result:
- report modules
- purpose of each module
- writing order
- missing-materials checklist

### `copywriting-brief`
Use when the main angle is already chosen and the next job is to prepare a reader-facing writing route.

Best for:
- recommendation article briefs
- public-facing post briefs
- editorial guidance
- article guidance
- writing route preparation
- title direction
- opening hook
- middle-point planning
- ending landing point

Typical result:
- target reader
- reader problem
- main angle
- title direction
- opening hook
- three development points
- usable book materials

## How To Use

Ask for the decision you need help with.

Examples:

- "Help me brainstorm the best angle for recommending this book."
- "Help me compare several reader-facing hooks for this title."
- "Help me create a proposal plan for this internal proposal report."
- "Help me organize this proposal into sections and identify missing evidence."
- "Help me turn this chosen angle into a writing brief for a recommendation article."
- "Help me turn this into editorial guidance for a public-facing article."

## Working Philosophy

- Think before writing
- Compare options before choosing
- Organize reports around persuasion, not information dumps
- Stay specific
- Avoid generic AI language
- Never invent proof

## File Layout

- `skills/using-superpowers/`

## Status

It does **not** yet include:
- final copy drafting
- manuscript reviewing
- polishing / rewriting
- campaign execution workflows
